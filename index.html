About

Fluix is a cross-platform framework for desktop application developing in Python. It provides completely new, "fluent", way to creating flexible, beautiful programs for any platforms (Linux, MacOS, Windows).

Gorgeous appearance on all platforms

Advantages

The main goal of creating Fluix was desire to make development process easier and faster, and also application - flexible and cool. Fluix is not the solution to all problems, and for high performance programs it is better to use C++ or Java, but it is definitely available concurrent for such modern tools as Electron and Flutter.

Create your own unique UI based on WebView

Like Flutter or Electron, Fluix closely related to web technologies, but it has several advantages over them. For example, development on Flutter is conducted on proprietary programming language Dart, that used only in Flutter, when Fluix uses convenient and widespread Python. UI layouts creating in Fluix also more easy and understandable then in Flutter.

What about Electron, it`s main problem is layout creating on archaic, unsuitable HTML or necessity to use web frameworks us React or Vue leading to more resources usage. Fluix solve that problem with great layouting system based on Pages and Widgets. Developers may not have strong skills in html, css and js to create awesome applications.
